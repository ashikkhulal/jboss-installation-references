JBOSS EAP 7.1.1 Installation on RHEL 8.1:

1. Install updates:
    $ sudo yum update -y
2. Install JDK:
    $ sudo yum install java-1.7.0-openjdk -y
    $ java -version
    $ which java
3. Download JBOSS 7.1.1 in /usr/share directory:
    $ sudo su
    $ cd /usr/share/
    $ wget http://download.jboss.org/jbossas/7.1/jboss-as-7.1.1.Final/jboss-as-7.1.1.Final.zip
4. Unzip the package rename the output folder:
    $ unzip -q jboss-as-7.1.1.Final.zip
    $ mv jboss-as-7.1.1.Final jboss-as
5. Add the JBOSS group and user:
    $ groupadd jboss
    $ useradd -s /bin/bash -g jboss jboss
6. Change the ownership for jboss-as:
    $ chown -Rf jboss.jboss /usr/share/jboss-as
7. Now, navigate to /home/jboss/ and add some lines in .bash_profile file
    $ cd /home/jboss/
    $ vim .bash_profile

    And add following lines to the file save and exit (wq!):
        JAVA_HOME=/usr/lib/jvm/java-11-openjdk-11.0.11.0.9-0.el8_1.x86_64  
        export JAVA_HOME  
        PATH=$JAVA_HOME/bin:$PATH  
        export PATH
8. Repeat step 7 for root user as well:
    $ cd /root/
    $ vim .bash_profile

    And add following lines to the file save and exit (wq!):
        JAVA_HOME=/usr/lib/jvm/java-11-openjdk-11.0.11.0.9-0.el8_1.x86_64  
        export JAVA_HOME  
        PATH=$JAVA_HOME/bin:$PATH  
        export PATH
9. Copy the jboss-as-standalone.sh script file to /etc/init.d/jboss and cd into /etc/init.d/
    $ cp /usr/share/jboss-as/bin/init.d/jboss-as-standalone.sh /etc/init.d/jboss
    $ cd /etc/init.d/
10. Once the directory is changed, edit the jboss file by adding the following lines below Export JAVA_HOME
    $ vim jboss
    And add following below export JAVA_HOME and save and exit (wq!)
        JBOSS_USER=jboss
        export JBOSS_USER
11. Next step is set permission for the jboss file
    $ chmod 755 jboss
12. Add systemd file for jboss in /etc/systemd/service
    $ cd /etc/systemd/service
    $ vim jboss.service
    Add the following lines and save and exit the file (wq!):
        [Unit]
        Description=Jboss Application Server
        After=network.target

        [Service]
        Type=idle
        Environment=JAVA_HOME=/usr/lib/jvm/java-11-openjdk-11.0.11.0.9-0.el8_1.x86_64 JBOSS_HOME=/usr/share/jboss-as JAVA=JAVA_HOME=/usr/lib/jvm/java-11-openjdk-11.0.11.0.9-0.el8_1.x86_64/bin/java
        User=jboss
        Group=jboss
        ExecStart=/usr/share/jboss-as/bin/init.d/jboss-as-standalone.sh
        TimeoutStartSec=600
        TimeoutStopSec=600

        [Install]
        WantedBy=multi-user.target
        
12. Start the JBoss service as follows:
    $ service jboss start
13. Now, stop the JBoss service:
    $ service stop jboss
14. Change directory to /usr/share/jboss-as/standalone/configuration/
    $ cd /usr/share/jboss-as/standalone/configuration/
15. And then edit the standalone.xml file. Open the file as follows:
    $ vim standalone.xml
    Once you are in the standalone.xml file, change the bind address in the public interface as follows. Save it and exit (wq!) after the appropriate changes are made.
    Change it from 127.0.0.1 to 0.0.0.0
